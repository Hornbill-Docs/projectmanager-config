# Overview 
Project Manager allows Project Managers and stakeholders to raise and manage programs and projects. Bring together scheduling, resources, project plans, tasks, costs, and much more to plan and manage your projects. With the use of the Hornbill's Business Process Manager, build projects that follow anything from a simple workflow to an industry-standard Project Management processes.

## Features

* **Portfolio**. Portfolio Management allows the management of multiple Programs, each containing multiple projects. If you have the Program Manager role, you will have visibility of all programs.
* **Projects**. The project portfolio provides a list of the active projects that you are a stakeholder in. Projects where you are not either the Project Manager or a Stakeholder will not be available from this view unless you have the Project Portfolio Manager role which grants visibility to all projects in the portfolio.
* **Resources**. Project Resources provides any user with the Project Portfolio Manager role the ability to view both individual and team workloads and capacity across all project tasks within the Project Manager application.
