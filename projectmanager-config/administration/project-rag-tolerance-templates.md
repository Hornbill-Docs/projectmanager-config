# Project RAG Tolerance Templates
Introduction Create default values for red, amber and green rag tolerance descriptions for each rag tolerance type. 
## Creating Project RAG Tolerance Templates

Select the **+** icon to create a new template

* RAG Tolerance - Select the rag tolerance you wish to define a template for
* Red Description - Enter the description of a red status for the specified rag tolerance
* Amber Description - Enter the description of an amber status for the specified rag tolerance
* Green Description - Enter the description of a green status for the specified rag tolerance
* Status - The status of the template

Select **Create** to add.

Repeat this process as needed to add more rag tolerance templates.

Once created, the values for red, amber and green description will automatically be pulled through when a new rag tolerance is added to a project.

## Managing Project RAG Tolerance Templates

Edit and remove templates, by selecting either the **Pencil** or **Trash Can** icons next to the template you wish to edit or remove.

* Removing a template will not affect existing Project rag tolerances
